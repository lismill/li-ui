## Li-UI
Generate a library for vue.js2，因为使用了 `highlight.js` 所以建议 `node >= 14.0.0`

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
### Package components
```
yarn lib
```

### Lints and fixes files
```
yarn lint
```
